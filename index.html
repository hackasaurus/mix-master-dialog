How Do Web Pages Work?

The Blueprint

This is what every page on the Web comes from: plain-text source code that anyone can read. It's written in a simple language called the Hypertext Markup Language, or HTML.

It's like a blueprint.
<div>

  
</div>





  
The Skeleton

  
A Web browser reads the HTML blueprint and builds a hierarchic structure that represents its understanding of the document.

This structure is like the skeleton of the page, and it's sometimes called the Document Object Model, or DOM.

  





  
The Final Product

  
Code written in a programming language called JavaScript serves as the page's muscles and nervous system; styling information written in a language called Cascading Style Sheets, or CSS, makes up its skin and clothing.

The result is a page that can be seen, heard, and even interacted with. It's like magic ink.
